# Git 分支規範與功能規劃

## 主分支
| 分支 | 用途 |
| :--- | :--- |
| `main` | 最終穩定展示版本 |
| `develop` | 全組開發整合版本 |

## 功能分支
| 分支名稱 | 負責內容 |
| :--- | :--- |
| `feature/drone-system` | 無人機移動、攻擊、AI |
| `feature/map-system` | 地圖、地形、場景 |
| `feature/mission-system` | 關卡、任務流程 |
| `feature/ground-defense-system` | 防空炮、雷達、飛彈 |
| `feature/ui-system` | RTS介面、小地圖 |
| `feature/swarm-ai` | 蜂群AI |
| `feature/electronic-warfare` | 電子干擾系統 |
| `feature/effects-system` | 爆炸、飛彈、雷射特效 |
